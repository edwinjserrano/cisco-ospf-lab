# Cisco OSPF Lab

Este repositorio documenta un laboratorio de OSPFv2 e IPv6 OSPFv3 en Cisco IOS.

## Características
- R1 ↔ R2 en área 0.
- R2 ↔ R3 en área 1 (OSPFv2).
- R2 como ABR entre área 0 y área 1.
- OSPFv3 habilitado en todas las interfaces con área 0.
- Passive-interface en LANs para evitar adyacencias innecesarias.

# Cisco OSPF Lab (EN)

This repository documents an OSPFv2 and IPv6 OSPFv3 lab in Cisco IOS.

## Features
- R1 ↔ R2 in area 0.
- R2 ↔ R3 in area 1 (OSPFv2).
- R2 acts as ABR between area 0 and area 1.
- OSPFv3 enabled on all interfaces in area 0.
- Passive-interface on LANs to avoid unnecessary adjacencies.
